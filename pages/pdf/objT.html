object和embed

2018年2月6日10:38:22
1、Object

HTML <object> 标签，几乎所有主流浏览器都拥有部分对 <object> 标签的支持。定义一个嵌入的对象。请使用此元素向您的 XHTML 页面添加多媒体。此元素允许您规定插入 HTML 文档中的对象的数据和参数，以及可用来显示和操作数据的代码。

<object> 标签用于包含对象，比如图像、音频、视频、Java applets、ActiveX、PDF 以及 Flash。

object 的初衷是取代 img 和 applet 元素。不过由于漏洞以及缺乏浏览器支持，这一点并未实现。

浏览器的对象支持有赖于对象类型。不幸的是，主流浏览器都使用不同的代码来加载相同的对象类型。

而幸运的是，object 对象提供了解决方案。如果未显示 object 元素，就会执行位于 <object> 和 </object> 之间的代码。通过这种方式，我们能够嵌套多个 object 元素（每个对应一个浏览器）。

  • 1、data(url) 规定对象使用的资源的 URL。
  • 2、name 为对象定义唯一的名称（以便在脚本中使用）。
  • 3、width 定义对象的宽度。
  • 4、height 定义对象的高度。
  • 4、type 定义被规定在 data 属性中指定的文件中出现的数据的 MIME 类型。

PS：检查flash是否是最新的

2、embed

标签定义嵌入的内容，比如插件。

  • 1、type 设置或返回 embed 元素中 type 属性的值。
  • 2、src 设置或返回 embed 元素中 src 属性的值。
  • 3、width 设置或返回 embed 元素中 width 属性的值。
  • 4、height 设置或返回 embed 元素中 height 属性的值。

embed加载flash是所有浏览器都ok的

embed加载flash是所有浏览器都ok的,但是加载pdf文件在firfox上就不行了，加载音频在firfox和opera上都不行

0、总结

在实际项目中如果用原生html的话一般都是两个一起用，避免一些浏览器不支持的情况，加载flash貌似还好吧，但是加载pdf或者音频、视频都就会出问题。
